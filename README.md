# 🚗 Used Car Price Analysis (End-to-End Data Analytics Project)

## 📌 Project Overview

This project focuses on analyzing the used car market using data-driven techniques to extract meaningful business insights.

The analysis is based on Spinny-like dataset and aims to understand pricing trends, depreciation patterns, demand variations, and identify best-value cars for customers and businesses.

---

## 🎯 Business Objectives

- Analyze price variation across different car brands  
- Identify best-value cars (low price + good condition)  
- Understand impact of factors like:
  - Year
  - Kilometers driven
  - Fuel type
  - Transmission
  - Location  
- Support pricing strategy and inventory decisions  

---

## 📊 Dataset Overview

- **Source**: Spinny-based used car dataset  
- **Total Records**: 1820  

### Key Features:
- Car Brand  
- Model  
- Manufacturing Year  
- Price  
- Kilometers Driven  
- Fuel Type  
- Transmission  
- City / Location
